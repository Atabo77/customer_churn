# Customer Churn Prediction

## Overview
This project predicts whether a customer is likely to leave a company (churn) based on their usage data. It helps businesses identify at-risk customers and take action to improve retention.

## Features
- Data preprocessing and handling categorical variables  
- Classification model using Scikit-learn  
- Interactive Streamlit app for making predictions  

## Tech Stack
Python | Pandas | Scikit-learn | Streamlit | GitHub
